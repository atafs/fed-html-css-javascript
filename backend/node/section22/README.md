# fed-html-css-javascript
Basics on **backend in nodeJS**
##

## Run a script in nodeJS
- Like most languages, a **js script** can be ran by typing:
> `node script.js`

## Install Packages
- In the terminal:
> `npm install <package>`

## Import into the project
- In the script add on top:
```javascript
var pkg = require("package_name");
```
